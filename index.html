инженер программист на работу с веб интерфесами

Вам предлагается средствами JavaScript сделать модальное окно, в котором будет две зоны:

  • Зона фильтра - тут для каждого поля (столбца) таблицы выводятся lable и select, в котором выбираются уникальные значения выбранного поля
  • Зона таблицы - просто красиво выводится вся таблица. Если она будет еще и с работающим пагинатором, то это огромный плюс

У вас есть генератор таблиц в этом коде со случайными данными (генерируется объект с ключами headers и data). Вам необходимо, чтобы по кнопке вызывалось модальное окно, в котором бы пришедшие данные автоматом раскидывались бы на две области.

Мы хотим увидить красивое решение в коде и красивый интерфейс

Видео, показывающее часть наработок и с чем вам придется столкнуться, можно посмотреть ТУТ

Ограничения: нет jquery, bootstrap или что-то подобное. Только голый javaScript свой CSS (можно sass). Вам НЕ НУЖНО связывать фильтры и таблицу.

Если вы сможете с этим справиться, то, скорее всего, процентов 95, что попадете в нашу команду
